CSS 변수 (variable) -> 변할 수 있는 값
일반적으로 변수는 :root { } 에 선언
선언 방식은 --변수이름: 값;
변수를 사용하기 위해서는 var(--변수이름);

변수 사용 예시
- #34DEF0 처럼 복잡한 값일 경우 계속 쓰다가 실수 할 수 도 있기 때문에 "반복되는 값"을 변수로 정의하면 실수를 줄일 수 있다

변수에 대한 값이 결정되지 않은 상황에서 미리보기 같은 느낌으로 미리 값을 설정해 놓을 수 있음 -> 폴백(fallback)
폴백으로 설정한 값은 실제 값이 아니라 임시로 설정된 값이기 때문에 저장되지는 않음

변수 값으로 숫자를 지정하고 이 값을 width나 height 등에 단위(px, em 등)를 붙여서 사용한 경우
--b-size: 200;
width: var(--b-size)px;
-> width: 200 px;
-> calc() 함수 사용 (계산 함수)

변수에는 반드시 순수한 값만 들어있어야 함
200px (X) -> 200 (O)

이미지를 넣을 때에는 url() 함수를 포함한 값을 넣어야 함
url("img/galaxy.jpg")

특정 범위 내에서 루트 변수에서 선언한 변수를 재정의 한 경우 -> 해당 변수가 삭제가 됨
-> 따라서 재정의를 하는 것이 아닌 새로운 이름의 변수를 만들어서 사용
